# Портфолио геймдизайнера

Это персональный сайт-портфолио, разработанный для демонстрации проектов, исследований и резюме геймдизайнера. Проект был переработан с использованием современных инструментов для улучшения производительности и удобства разработки.

## Технологический стек

- **Сборщик**: [Vite](https://vitejs.dev/) - быстрый и современный инструмент для сборки фронтенда.
- **Стили**: [Sass](https://sass-lang.com/) - CSS-препроцессор для более чистого и структурированного кода стилей.
- **JavaScript**: Современный JavaScript (ES6+ Modules) для интерактивности.
- **Линтинг и форматирование**:
  - [ESLint](https://eslint.org/) - для анализа и исправления ошибок в JavaScript.
  - [Stylelint](https://stylelint.io/) - для анализа и исправления ошибок в SCSS.
  - [Prettier](https://prettier.io/) - для автоматического форматирования кода.

## Структура проекта

Все исходные файлы находятся в директории `src/`. Готовый для развертывания сайт собирается в директорию `dist/`.

```
│   ├── js/
│   │   ├── components/

```

## Как начать работу

### Требования

- [Node.js](https://nodejs.org/) (версия 18.x или выше)
- [npm](https://www.npmjs.com/) (обычно поставляется с Node.js)

### Установка зависимостей

1.  Клонируйте репозиторий:

    ```bash
    git clone https://github.com/silenceremember/silenceremember.github.io.git
    cd silenceremember.github.io
    ```

2.  Установите все необходимые зависимости:
    ```bash
    npm install
    ```

### Основные команды

- **Запуск сервера для разработки**:
  Запускает локальный сервер с автоматической перезагрузкой при изменениях в файлах.

  ```bash
  npm run dev
  ```

  Сайт будет доступен по адресу `http://localhost:5173/` (порт может отличаться).

- **Сборка проекта для продакшена**:
  Собирает, оптимизирует и минифицирует все файлы в директорию `dist/`.

  ```bash
  npm run build
  ```

- **Предпросмотр сборки**:
  Запускает локальный сервер для просмотра собранной версии из папки `dist/`.
  ```bash
  npm run preview
  ```

### Линтинг и форматирование

- **Проверить JavaScript**:

  ```bash
  npm run lint:js
  ```

- **Проверить стили (SCSS)**:

  ```bash
  npm run lint:css
  ```

- **Отформатировать весь код**:
  ```bash
  npm run format
  ```
